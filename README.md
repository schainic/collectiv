# Collectiv
A group payment system made possible by Capital One's API

## Features
- Complete sign up and login system
- Add groups
- Add members to groups
- Add funds to groups
- Deposit money into a fund for a group

## Stack
#### Front end
One page React.js web application and SCSS
#### Back end
Node.js and MongoDB
